# TodoList

## Styled Component 
[styled-component-정리](./STYLED.md)
## 구성 요소

- TodoTemplate: TodoTemplate를 담는 카드

- TodoHead : TodoTemplate 카드에 들어가는 Head

- TodoList : 할일이 몇개가 들어가는지 알려주는 요소

- TodoItem : 해야할 일들을 적는 컴포넌트

- TodoCreate : 해야할 일을 작성하는 +버튼

```js
    <>
      <GlobalStyle />
      <TodoTemplate>
        <TodoHead />
        <TodoList />
        <TodoCreate />
      </TodoTemplate>
    </>
```
